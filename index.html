关于自动化操作

安卓：auto.js、一触即发

ios：快捷指令（Shortcuts）：这是由苹果官方提供的应用程序，可以创建自定义的快捷指令来简化和自动化常见任务。它具有直观的用户界面和强大的功能。 Scriptable：这是一款专为 JavaScript 开发者设计的应用程序。它允许你使用 JavaScript 编写自定义的脚本，并将其用于自动化各种任务和操作。 Pythonista：如果你熟悉 Python 编程语言，Pythonista 是一个很好的选择。它提供了一个完整的 Python 编程环境，让你可以编写和运行自定义的脚本，实现各种自动化任务。 Workflow：Workflow 是一款强大的自动化应用程序，可以通过拖放操作来创建复杂的工作流程。它可以自动执行多个任务和操作，并与其他应用程序集成。 请注意，这只是一些在iOS平台上常见的自动化脚本软件之一。对于每个应用程序，我建议你阅读其相关信息、用户评价和隐私政策，以确定是否符合你的需求和个人偏好。 另外，随着时间推移，新的自动化脚本软件可能会出现，因此建议你保持关注并进行适当的研究。
